Peta Rasio Jenis Kelamin

Peta Rasio Jenis Kelamin adalah peta yang memuat informasi perbandingan jumlah laki-laki dengan perempuan di suatu wailyah.

Peta Rasio Jenis Kelamin memuat informasi sebagai berikut:

  • Batusitanduk: laki-laki = 1170 / perempuan = 1173
  • Harapan: Laki-laki = 1126 / perempuan = 1162
  • Kalibamamase: Laki-laki = 661 / perempuan = 652
  • Saragi: Laki-laki = 523 / perempuan = 524
  • Walenrang: Laki-laki = 891 / perempuan = 939
  • Barammamase: Laki-laki = 1340 / perempuan = 1333
  • Kelurahan Bulo: Laki-laki = 979 / perempuan = 971
  • Lalong: Laki-laki = 1441 / perempuan = 1444
  • Tombang: Laki-laki = 961 / perempuan = 1034
